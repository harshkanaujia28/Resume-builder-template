# Resume Builder Template

A frontend-only resume builder template built with Next.js.

## Features
- Modern resume builder UI
- Structured resume sections (education, experience, skills)
- ATS-friendly layout
- Fully responsive design
- Easy to customize branding and content

## Tech Stack
- Next.js
- React
- Tailwind CSS
- Framer Motion

## Not Included
- Backend / API
- Authentication
- AI integration
- Real PDF export

## Installation
npm install
npm run dev

## Deployment
Deploy easily on Vercel.
